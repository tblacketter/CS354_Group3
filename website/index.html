What Is the F* Programming Language?
F* is a dependently typed functional programming language designed for program verification. It empowers developers to write code with mathematically provable guarantees—ranging from safety and correctness to advanced security properties.

At its core, F* blends expressive types, functional programming, and automated theorem proving to help you reason about your software with precision.
Why Learn F*?
F* is more than a programming language—it’s a tool for writing provably correct software.

Whether you’re building secure systems, exploring formal verification, or advancing your research, F* gives you the power to write code that you can trust, with guarantees backed by mathematical proofs.
