Installation Instructions
There are binaries available for Windows, Arm64, and Linux on the F* Github Page

F* Github

  • Download the binary for your system.

  • Add the path for fstar and Z3 to your path

  • Now you should be able to verify F* files

The binary allows for verification. If you want to contribute to the project or bootstrap/compile F* using Ocaml, follow the detailed steps listed here:

  • Z3 version 4.8.5
  • Z3 version 4.13.3
